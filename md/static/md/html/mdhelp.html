Markdown guide

Headers

# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag

Bold

**This text will be bold**
__This will also be bold__
**You can combine it with _italic_**

Italic

*This text will be italic*
_This will also be italic_
_You can combine it with **bold**_

Lists

Unordered

* Item 1
* Item 2
* Item 2a
* Item 2b

Ordered

1. Item 1
2. Item 2
3. Item 3
* Item 3a
* Item 3b

Links

http://www.rouxcode.ch - automatic!

Would become:
http://www.rouxcode.ch

email@address.org - automatic (spam protected)!

Would become:
email@address.org

[Developer](http://www.rouxcode.ch)

Would become:
Developer

Tables

You can create tables by assembling a list of words and dividing them with hyphens - (for the first row), and then separating each column with a pipe |:

First Header | Second Header
------------ | -------------
Content from cell 1 | Content from cell 2
Content in the first column | Content in the second column

Would become:

First Header Second Header
Content from cell 1 Content from cell 2
Content in the first column Content in the second column

Strikethrough

Any word wrapped with two tildes (like ~~this~~) will appear crossed out.
